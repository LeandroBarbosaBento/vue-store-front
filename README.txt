# Iniciar o projeto

Para iniciar o projeto basta rodas os seguintes comandos no terminal estando na raiz do projeto:

```bash
npm install
npm run serve
```

### Observar se a url do arquivo "src/utils/api.js", na linha 7, é o mesmo no qual o servidor que o backend do projeto está rodando.
